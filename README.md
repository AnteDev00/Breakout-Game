# Breakout

My attempt at recreating legendary 2d arcade game in C++ using only SDL and helper libraries.

## Key principles used:
- Object Oriented aproach
- Using SOLID priciples
- Using fundemental game components

## Features
- 2D Phyiscs
- UI 
- Level System (via XML files)
- etc.

## Libraries used:
- **SDL2**       
- **SDL2_image** 
- **SDL2_mixer** 
- **SDL2_ttf**   
- **TinyXML2**
